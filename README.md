# Skin Disease Screening System 🩺📱

An AI-driven mobile application designed for the early detection and continuous tracking of skin conditions. This project was developed as part of the Mobile Device Application Development course at Walailak University.

## 🌟 Key Features
* **Camera & Gallery Integration:** Capture a new photo or upload an existing image using `expo-image-picker`.
* **Simulated AI Analysis:** Provides an instant simulated risk assessment, including Risk Levels and Confidence Scores.
* **Local Offline Database:** Utilizes `expo-sqlite` to securely store user scan histories directly on the device.
* **Full CRUD Operations:** * **Create:** Save new scan results.
  * **Read:** View chronological scan history.
  * **Delete:** Remove unwanted records via long-press gestures.
* **Health Education:** Built-in medical guidelines and advice for common skin diseases.

## 🛠️ Tech Stack
* **Frontend:** React Native, Expo
* **Navigation:** React Navigation (Stack & Bottom Tabs)
* **Database:** SQLite (`expo-sqlite`)
* **Icons:** `@expo/vector-icons`
